# Antiderivative
Antiderivative is reverse (undoing) of the [[Derivatives]]

$F$ is antiderivative of $f$ if 
$$F'(x) = f(x)$$

If $F$ is an antiderivative, then $F(x) + c$ is also antiderivative (C is a constant). This is just raising the graph up by $C$
